# AtomCore
AtomCore is a lightweight, all-in-one core plugin for SMP servers. It features a fully customizable GUI Shop, built-in Economy, flicker-free Scoreboard, TPA, Home systems, and Dual Language (EN/TR) support. Compatible with Paper, Spigot, and Purpur (1.20-1.21) without needing external plugins like Vault. The ultimate SMP essential!
